# SoundToText

## Description
SoundToText is an application for converting sound to text. It uses advanced speech recognition technologies to accurately and quickly convert audio files into text format.

## Installation
1. Ensure you have Python 3.8 or higher installed.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Run the application:
   ```bash
   python main.py
   ```
2. Follow the on-screen instructions to upload audio files and get the text result.

## Requirements
- Python 3.8+
- Libraries listed in `requirements.txt`

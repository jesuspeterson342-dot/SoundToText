# SoundToText

## Описание
SoundToText - это приложение для преобразования звука в текст. Оно использует передовые технологии распознавания речи для точного и быстрого преобразования аудиофайлов в текстовый формат.

## Установка
1. Убедитесь, что у вас установлен Python 3.8 или выше.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Использование
1. Запустите приложение:
   ```bash
   python main.py
   ```
2. Следуйте инструкциям на экране для загрузки аудиофайлов и получения текстового результата.

## Требования
- Python 3.8+
- Библиотеки из файла `requirements.txt`
